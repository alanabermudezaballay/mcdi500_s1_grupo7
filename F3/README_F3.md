# Fase 3 — Algoritmos, recursividad y complejidad

Esta carpeta contiene el avance de la Formativa 3 del proyecto grupal MCDI500.

## Objetivo

Reutilizar el dataset procesado en Fase 2 para implementar algoritmos estructurados, algoritmos recursivos y mediciones iniciales de complejidad temporal y/o espacial.

## Insumo de datos

El notebook utiliza:

`data/processed/dataset_procesado.csv`

Este archivo corresponde al dataset limpio y validado en Fase 2.

## Contenido

- `F3_Algoritmos_Complejidad.ipynb`: notebook central de Fase 3.
- `src/carga_validacion.py`: funciones de carga y validación inicial del dataset.
- `outputs/`: carpeta reservada para resultados de mediciones.

## Ejecución

1. Activar el entorno virtual.
2. Abrir el notebook `F3/F3_Algoritmos_Complejidad.ipynb`.
3. Seleccionar el kernel del entorno `.venv`.
4. Ejecutar todas las celdas.
5. Validar que no existan errores.
